Snakemake workflow: rna-count-kallisto

This is the Snakemake workflow for RNA-Seq read count, powered by Kallisto. Optional quality controls are performed by FastQC and aggregated by MultiQC

Each tool belong to their respective authors.

Currently under active development, this pipeline is not intended to be used.

See the embedded wiki for more information.
